# CLASS101 멤버십 리디자인 (클론 연습 프로젝트)

CLASS101 멤버십 서비스를 참고해 다크 + 네온 옐로우 테마로 재해석한 정적 웹사이트입니다.

## 페이지 구성
- `index.html` — 메인 페이지 (히어로, 추천/상황별/인기 클래스)
- `membership.html` — 멤버십 서브 페이지 (Swiper 히어로 슬라이더, 추천 멤버십, 카테고리별 멤버십, 정렬 드롭다운)
- `membership-baksaesama.html` — 브릉이(브르) 멤버십 상세 페이지
- `home-baksaesama.html` — 브르 크리에이터 홈 (프로필, 커뮤니티 글, 클래스)
- `payment-baksaesama.html` — 결제 페이지
- `login.html` — 로그인 페이지

## 폴더
- `css/` — `style.css`, `font.css`
- `image/` — 이미지 에셋

## 사용 기술
- HTML / CSS (Vanilla)
- [Swiper](https://swiperjs.com/) (히어로 슬라이더)
- 폰트: Paperozi(Paperlogy), Anton

> 학습/포트폴리오 목적의 클론 프로젝트이며, 일부 콘텐츠·이미지는 CLASS101에서 참고했습니다.
